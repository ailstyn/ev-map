# USA Energy Prices Map

An interactive Streamlit prototype comparing gas prices and electricity prices across the USA, visualized on an interactive map down to the finest available resolution.

## Features

- Interactive choropleth map showing state-level price data
- Toggle between gas and electricity price views
- Real-time data from public APIs (EIA, AFDC)
- Easy data aggregation and visualization

## Setup

### Requirements

- Python 3.8+
- Streamlit
- Plotly
- Folium
- Pandas
- Requests

pip install -r requirements.txt
